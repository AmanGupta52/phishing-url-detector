# Phishing URL Detection Using Machine Learning (Mini SOC Tool)

A machine learning-based web application to detect phishing URLs in real time using URL feature extraction and Random Forest classification.

## Features
- Real-time URL analysis
- Machine Learning based detection
- Streamlit web interface
- SOC-style workflow simulation

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas
